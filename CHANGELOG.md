# Changelog

All notable changes to the AI Transformation Lab are recorded here.

## v1.0.0 — Initial release

**Released:** April 2026

### What's included

**Program structure**
- Session 1: Foundation + problem framing (60–90 min)
- Session 2: Applied lab + prompt mastery (60–90 min)
- Session 3: Showcase + champion toolkit (60–90 min)
- Session 4: Builder track — optional (90 min+), self-selected

**Features**
- Group session and self-paced modes with content that adapts to each
- Prompt lab with RCTFX framework (Role, Context, Task, Format, Constraints)
- Wins wall with real-time cohort logging and presentation mode
- Facilitator notes per session with session timer
- ROI capture and JSON export
- Cross-functional prompt patterns (Finance, Supply Chain, Strategy, Operations)
- AI tool decision tree
- Failure modes guide with fixes
- Worked examples per session (collapsible)
- "Do this now" action callouts throughout
- Builder track: workflow designer, UI prototyper, vibe coding intro
- Guardrails section with function-specific guidance
- Connect tab with facilitator contact and feedback form
- Web Share API for mobile sharing
- Offline-capable after first load
- Fully responsive — desktop and mobile
- Single self-contained HTML file, no install required

**Technical**
- No external dependencies except Google Fonts (degrades gracefully offline)
- All event binding via addEventListener (no onclick attributes)
- Universal clipboard fallback for file:// and restricted contexts
- Mobile-safe JSON export with clipboard fallback
- Open Graph meta tags for social sharing preview
